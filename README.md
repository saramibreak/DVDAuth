# DVDAuth
This is CSS or CPRM authentication tool

## Usage
        DVDAuth.exe <DriveLetter> <ProtectType> <OutFile>
        ProtectType: css or cprm

## Development Tool
 Visual Studio 2017 (Visual C++ 2017)  
 Windows Driver Kit (WDK) 7.1
